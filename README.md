# Nexter — React + TypeScript + Vite

A migration of the Nexter CSS Grid layout project to a modern React + TypeScript stack, built with Vite and deployed to GitHub Pages via GitHub Actions.

**[Live Demo](https://atraineedeveloper.github.io/Nexter-react/)**

## Tech Stack

- React 19
- TypeScript
- Vite
- SCSS Modules
- GitHub Actions (CI/CD)
- GitHub Pages (hosting)

## Getting Started

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deploy

Deployment is automatic — every push to `master` triggers the GitHub Actions workflow that builds and deploys to GitHub Pages.
